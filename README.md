# Epoch Talents
Web based Talent calculator for the World of Warcraft private server Project Epoch

### Works best in Chromium based Browsers (Chrome, Opera, Brave, Edge)
### This Project is still work in Progress and not all Classes are implemented yet
The following Classes are available:
- Rogue
- Shaman
- Hunter
- Warrior
- Paladin
- Druid
- Priest

The following Classes are still missing:
- Warlock
- Mage
